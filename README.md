# Steaph API

## Requirements

* Docker (Optinal)
* Mongo v2.6.10 in :27017
* Node.js v6.5

## Target environemnt

* Start up Steaph API
```
$ docker-compose up
```

## Development environment

* Start up TypeScript Compiler
```
$ mongod
```

* Start up TypeScript Compiler
```
$ npm install
$ npm run babelw
```

* Start up Steaph API
```
$ nodemon
```
